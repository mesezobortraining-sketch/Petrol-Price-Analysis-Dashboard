# Petrol Price Analysis Dashboard

![Petrol Price Dashboard](dashboard.png)

## Project Overview

This project analyzes petrol prices across different countries and regions using **Excel** and **Power BI**. The goal is to understand how petrol prices vary globally and present the insights in a simple interactive dashboard.

## Dataset

The dataset contains petrol price information for **101 countries across 8 regions**. It includes petrol price per liter, price per gallon, price change percentage, country, region, and price level.

## Dashboard Summary

The Power BI dashboard shows:

- Average petrol price per liter
- Average petrol price per gallon
- Highest and lowest petrol price
- Petrol prices by region
- Petrol prices by country
- Petrol price level categories
- Global map of petrol price distribution

## Key Insights

- The dataset covers **101 countries** and **8 regions**.
- The average petrol price is about **$1.38 per liter**.
- The average petrol price is about **$5.23 per gallon**.
- Countries such as **Hong Kong, Singapore, Netherlands, Denmark, and Germany** have some of the highest petrol prices.
- Petrol prices vary significantly across regions.

## Conclusion

This project helps users understand global petrol price differences through a clear Power BI dashboard. It is useful for comparing petrol prices by country, region, and price level.
